## 🚀 CÓMO EJECUTAR

### 1. Instalar Dependencias
```bash
npm install
cd api && npm install && cd ..
```

### 2. Ejecutar Backend (Terminal 1)
```bash
cd api
node server.js
```

### 3. Ejecutar Frontend (Terminal 2)
```bash
npm run dev
```

### 4. Abrir en Navegador
```
http://localhost:3000
```
